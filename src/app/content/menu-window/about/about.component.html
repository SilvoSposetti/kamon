Kamon

This is a browser start-page made using the Angular framework.

It features a search bar with custom shortcut recognition, combining a bookmark-like functionality accessible directly with the keyboard.

The background of the application uses an HTML canvas element which can render various animations to the screen.

This project started in September, 2017 and, as it is a side project for me, I usually improve from time to time, when I have the time

Additional information can be found on Github.
